# Type-Ahead
A web app that displays suggestions while a user is entering text to a input field.

# Uses
HTML5, CSS3, Modern Javascript(ES2015+), Node.js, MongoDB

# Usage
$ mongod, $ npm install,$ node seed.js,$ node index.js
